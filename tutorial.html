Tutorial: jQuery e Bootstrap 4

Nesta página, vamos explorar alguns componentes e funcionalidades essenciais do Bootstrap 4 e do jQuery, que são fundamentais para o desenvolvimento web moderno. Aprenderemos sobre o Sistema de Grade e Cards do Bootstrap, e os métodos toggle() e css() do jQuery.

Componentes Bootstrap 4

1. Sistema de Grade (Grid System)

O Sistema de Grade do Bootstrap permite criar layouts de página responsivos e flexíveis, dividindo o espaço disponível em até 12 colunas. As colunas se reorganizam automaticamente dependendo do tamanho da tela do dispositivo, garantindo que seu site seja visualmente agradável em telefones, tablets e desktops. Se você não quiser usar todas as 12 colunas individualmente, pode agrupá-las para criar colunas mais amplas.

O Bootstrap 4 possui cinco classes de grade que definem o comportamento das colunas em diferentes tamanhos de tela:

  • .col- (dispositivos extra pequenos - telas com largura menor que 576px)
  • .col-sm- (dispositivos pequenos - telas com largura igual ou maior que 576px)
  • .col-md- (dispositivos médios - telas com largura igual ou maior que 768px)
  • .col-lg- (dispositivos grandes - telas com largura igual ou maior que 992px)
  • .col-xl- (dispositivos extra grandes - telas com largura igual ou maior que 1200px)

As classes acima podem ser combinadas para criar layouts mais dinâmicos e flexíveis. Para usar, você cria uma linha (`

`). Em seguida, adicione o número desejado de colunas (tags com classes `.col-*-*` apropriadas). O primeiro asterisco (*) representa a responsividade (sm, md, lg ou xl), enquanto o segundo representa um número, que deve somar 12 para cada linha.

2. Cards

Um Card no Bootstrap 4 é uma caixa com bordas e um espaçamento em torno de seu conteúdo. Ele inclui opções para cabeçalhos, rodapés, conteúdo, cores, etc. No Bootstrap 3, cards substituem os panels, wells e thumbnails.

Um card básico é criado com a classe .card e o conteúdo dentro do card que tem uma classe .card-body.

Funções jQuery

1. Método toggle()

O método toggle() do jQuery é utilizado para alternar entre ocultar e mostrar um elemento. Os elementos visíveis são ocultados e os elementos ocultos são mostrados.

A sintaxe é: $(selector).toggle(speed, callback);.

  • O parâmetro opcional speed especifica a velocidade da ocultação/exibição e pode assumir os valores: "slow", "fast" ou milissegundos.
  • O parâmetro opcional de retorno de chamada (callback) é uma função a ser executada após a conclusão do toggle().

2. Método css()

O método css() do jQuery é utilizado para definir ou retornar o valor de uma ou mais propriedades CSS dos elementos selecionados.

Para retornar o valor de uma propriedade CSS especificada, use a sintaxe: $(selector).css("propertyname");. Para definir uma propriedade CSS especificada, use a sintaxe: $(selector).css("propertyname", "value");. Para definir várias propriedades CSS, use a sintaxe: $(selector).css({"propertyname": "value", "propertyname": "value", ...});.
